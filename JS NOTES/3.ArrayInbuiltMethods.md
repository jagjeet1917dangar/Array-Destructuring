# JavaScript – Common Array Methods

---

## 1. The `slice()` Method

* `slice()` ek **non-destructive** method hai, yani original array change nahi hota.
* ye ek **portion copy** karke naya array return karta hai.

```js
let arr = [10, 20, 30, 40, 50];
let part = arr.slice(1, 4);
console.log(part); // [20, 30, 40]
console.log(arr);  // [10, 20, 30, 40, 50]
```

* 1st parameter = start index
* 2nd parameter = end index (exclusive)
* agar end nahi likha, to end tak copy karega.

---

## 2. The `splice()` Method

* `splice()` **array ko modify** karta hai (original array change hota hai).
* use hota hai element **add / remove** karne ke liye.

```js
let arr = [1, 2, 3, 4, 5];
arr.splice(2, 1, 99, 100);
console.log(arr); // [1, 2, 99, 100, 4, 5]
```

* pehla argument → starting index
* doosra → kitne elements delete karne hain
* baad wale → new elements jo insert karne hain

---

## 3. The `concat()` Method

* Do ya zyada arrays ko combine karta hai.
* Original array change nahi hota, naya array milta hai.

```js
let a = [1, 2];
let b = [3, 4];
let c = a.concat(b);
console.log(c); // [1, 2, 3, 4]
```

* strings ya single values bhi add kar sakte ho.

---

## 4. The `join()` Method

* Array ke elements ko ek single string me convert karta hai.
* Default separator comma hota hai, but hum custom bhi de sakte hain.

```js
let arr = ["apple", "banana", "mango"];
console.log(arr.join(" - ")); // apple - banana - mango
```

* useful when you want readable output.

---

## 5. The `indexOf()` Method

* kisi element ka **first index** return karta hai.
* agar element nahi mila to `-1` return hota hai.

```js
let arr = [5, 10, 15, 10];
console.log(arr.indexOf(10)); // 1
```

* case-sensitive hota hai (for strings).

---

## 6. The `lastIndexOf()` Method

* ye bhi same hai but **last occurrence** ka index return karta hai.

```js
let arr = [5, 10, 15, 10, 20];
console.log(arr.lastIndexOf(10)); // 3
```

---

## 7. The `includes()` Method

* check karta hai ki element array me hai ya nahi.
* returns true / false.

```js
let fruits = ["apple", "banana", "grapes"];
console.log(fruits.includes("banana")); // true
console.log(fruits.includes("mango"));  // false
```

---

## 8. The `sort()` Method

* elements ko sort karta hai, **but as strings by default**.
* matlab numbers me dikkat aa sakti hai.

```js
let arr = [10, 2, 30];
console.log(arr.sort()); // [10, 2, 30] → wrong order (string compare)
```

* Correct numeric sort ke liye:

```js
arr.sort((a, b) => a - b);
console.log(arr); // [2, 10, 30]
```

* ye compare function ascending / descending order decide karta hai.

---

## 9. Using `localeCompare()` for String Sorting

* jab strings me capital-small letter difference ho ya language-based sorting chahiye, tab use hota hai.

```js
let names = ["zara", "Aman", "harsh"];
names.sort((a, b) => a.localeCompare(b));
console.log(names); // ["Aman", "harsh", "zara"]
```

* ye alphabetical aur case-insensitive sorting me help karta hai.

---

### Thoda Recap:

* **slice()** – copy part (no change)
* **splice()** – add/remove (changes array)
* **concat()** – merge arrays
* **join()** – make string
* **indexOf() / lastIndexOf()** – find position
* **includes()** – check existence
* **sort()** – arrange data
* **localeCompare()** – advanced string sorting

---

**Note:** Slice aur splice dono naam similar lagte hain, par ek array todta hai aur ek modify karta hai — confuse mat hona.
