# JavaScript – Objects (Dot vs Bracket Notation, Methods, and Destructuring)

---

## 1. What is an Object?

* Object ek key–value pair structure hota hai.
* Basically ye real world cheezon ko represent karne ke kaam aata hai.
* Example:

```js
const person = {
    name: "Jagjeet",
    age: 20,
    city: "Ahmedabad"
};
```

* yaha `name`, `age`, aur `city` keys hain, aur unki values strings/numbers hain.

---

## 2. Accessing Object Values

### Dot Notation

```js
console.log(person.name);
console.log(person.age);
```

* Simple aur commonly used syntax.
* par dikkat ye hai ki **keys fix** hone chahiye (string literals allowed nahi hote).
* Example:

```js
let key = "city";
console.log(person.key); // ❌ undefined
```

### Bracket Notation

```js
console.log(person["name"]);
```

* Jab key dynamic ho ya variable me store ho tab use hoti hai.

```js
let key = "city";
console.log(person[key]); // ✅ Ahmedabad
```

* isme aap space ya special character wali keys bhi use kar sakte ho.

```js
const data = { "first name": "Jagjeet" };
console.log(data["first name"]); // works
```

**Note:** generally dot notation simple cases me aur bracket notation dynamic keys me use hoti hai.

---

## 3. Object Methods

* Methods wo functions hote hain jo object ke andar likhe jaate hain.
* Basically object ke behaviour ko define karte hain.

```js
const student = {
    name: "Karan",
    marks: 95,
    greet: function() {
        console.log("Hello " + this.name);
    }
};
student.greet(); // Hello Karan
```

* yaha `this` current object ko refer karta hai.
* short syntax me bhi likh sakte ho:

```js
const user = {
    name: "Rahul",
    sayHi() {
        console.log("Hi " + this.name);
    }
};
```

---

## 4. Adding and Deleting Properties

```js
person.country = "India"; // add
delete person.age;        // delete
```

* aise object ke properties change ya remove kar sakte ho.

---

## 5. Object Destructuring

* Destructuring ka matlab hota hai object ke properties ko directly variables me tod dena.

```js
const user = {
    name: "Aman",
    age: 22,
    city: "Delhi"
};

const { name, age } = user;
console.log(name); // Aman
console.log(age);  // 22
```

* agar variable ka naam change karna ho:

```js
const { city: location } = user;
console.log(location); // Delhi
```

* aur agar koi property exist nahi karti:

```js
const { hobby = "Cricket" } = user;
console.log(hobby); // Cricket
```

* nested destructuring bhi possible hai:

```js
const student1 = {
    info: {
        name: "Simran",
        roll: 45
    }
};
const { info: { name: studentName } } = student1;
console.log(studentName); // Simran
```

---

## 6. Summary (in short)

* Objects hold key–value pairs
* Dot notation – fixed keys ke liye
* Bracket notation – variable ya space wale keys ke liye
* Methods – function inside object (use `this`)
* Destructuring – shortcut to extract values

---

**Note:** Object JS ka backbone hai, isko acche se samjho kyunki arrays, functions, sab ek type ke object hi hain internally.
